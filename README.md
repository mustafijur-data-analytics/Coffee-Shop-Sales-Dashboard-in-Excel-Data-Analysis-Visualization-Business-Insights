# Coffee-Shop-Sales-Dashboard-in-Excel-Data-Analysis-Visualization-Business-Insights
This project presents a comprehensive Coffee Shop Sales Analysis Dashboard built in Microsoft Excel, designed to transform raw transactional data into actionable business insights. The dashboard visualizes key performance indicators (KPIs) such as total revenue, transaction count, average bill per person, and sales distribution across categories, sizes, locations, and time.

Using advanced Excel features like Pivot Tables, Charts, Slicers, and Data Cleaning techniques, this project demonstrates how data analysts can analyze customer behavior, identify peak hours, and optimize product performance.

☕ Peak sales occur during morning hours (7 AM – 10 AM)
🥐 Coffee category dominates sales (~39%)
📍 Hell’s Kitchen generates the highest revenue
📦 Barista Espresso is the top-selling product
📅 Weekdays outperform weekends in revenue
